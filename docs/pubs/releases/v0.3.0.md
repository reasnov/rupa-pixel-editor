# Release Notes: v0.3.0 (The Modular Barista)

This major release transforms the **Rupa Pixel Editor** into a professional-grade workspace with a completely refactored architecture, intuitive selection mechanics, and a refined HUD designed for serious craftsmanship.

## üöÄ Highlights

### 1. Architectural Evolution (SOC)

The engine has been entirely rebuilt following the **Separation of Concerns (SOC)** principle.

- **Modular State:** `EditorState` is now a root orchestrator delegating to specialized sub-states: `Canvas`, `Cursor`, `Palette`, `Studio`, `Project`, and `Selection`.
- **Service-Oriented Logic:** Core logic moved into dedicated services for Movement, Pixeling, Manipulation, Clipboard, and Persistence.

### 2. Enhanced Selection (Selection)

The "Digital Pixeling" workflow is now much more powerful:

- **Persistent Selection:** Active regions now stay visible after releasing the Shift key.
- **Marching Ants UI:** A dynamic, animated lasso border provides professional visual feedback.
- **Batch Fill:** Press `Space` while selecting to instantly fill the region with the active color.
- **Grouped History:** Paste, Cut, and Batch Fill operations are now grouped into single Undo/Redo entries.

### 3. Navigation & Precision

- **Go To Coordinate (G):** A new direct-jump feature allows you to move the Cursor to exact Cartesian coordinates.
- **Data Integrity:** Fully separated "Empty Pixels" from the "Studio Cream" color using `null` data values. You can now use `#eee8d5` as a valid color in your artwork.

### 4. HUD Overhaul

The interface has been restructured into specialized Bars for a cleaner workspace:

- **TopBar:** Centralized command center for project tools.
- **LeftBar:** Floating floating color palette.
- **RightBar:** Unified **Color Inspector** (Active & History) and Cursor Stats.
- **BottomBar:** Floating State Indicator with 20-second autohide.

### 5. Smart SVG Artifacts

The export engine now uses a **Greedy Rect-Merging** algorithm. It intelligently combines connected pixels of the same color into optimized paths, significantly reducing SVG file complexity and size.
