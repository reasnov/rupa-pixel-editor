# Release Notes: v0.5.0-alpha (The Swift Weaver)

This alpha release focuses on automation and efficiency for the artisan, introducing intelligent tools to reduce repetitive keyboard tasks. It also features a unified audio engine to ensure a seamless atmospheric experience.

### 1. Dye Soak (Flood Fill)

- **Shortcut**: `F`
- Automatically fill connected stitches of the same color starting from the Needle position. Perfect for large shapes and background work.

### 2. Fiber Bleach (Global Recolor)

- **Shortcut**: `Alt + R`
- Instantly replace every instance of the color under the Needle with your currently active Dye across the entire layer.

### 3. Spirit Pick (Magic Wand Selection)

- **Shortcut**: `W`
- Select all connected stitches of the same color. Once selected, you can manipulate the entire motif as a single unit.

### 4. Advanced Selection Lasso

- Selections are no longer limited to rectangles. The studio now supports arbitrary pixel-based selections (Motifs), rendered with a soft glow and marching ants.
