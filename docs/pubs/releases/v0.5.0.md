# Release Notes: v0.5.0 (The Spindle of Time)

This milestone release transforms **Rupa Studio** from a static pixel editor into a professional kinetic workshop. It introduces the **Spindle Engine** for motion editing, an architectural standard for CSS, and comprehensive UI/UX guidelines to ensure the studio remains a meditative sanctuary for artisans.

The studio now supports temporal sequences, allowing artisans to breathe life into their stitches.

### 1. Professional Linear Timeline

- **The Spindle**: A completely redesigned, modular timeline track.
- **Linear Track**: Frame blocks are now continuous segments whose width represents their duration.
- **Time Ruler**: Accurate millisecond markers to help time your animations.
- **The Playhead**: A magenta "Needle of Time" that indicates the active frame position.

### 2. Ghost Threads (Onion Skinning)

- **Visual Shadow**: Visualize up to 2 previous frames with graded transparency.
- **Shortcut**: `Alt + G` to toggle ghosts on/off.
- **Intuitive HUD**: A quick-toggle button is now available directly on the Spindle panel.

### 3. Kinetic Weaves (Video Export)

- **Advanced Export**: Create artifacts in **WebM**, **MP4**, **GIF**, and **Animated SVG** formats.
- **Safety Protocol**: Kinetic formats are automatically disabled for single-frame projects to prevent errors.
