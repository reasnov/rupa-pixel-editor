# Release Notes: v0.6.0 (The Logic of Loom)

Milestone ini memperkenalkan kecerdasan geometris ke dalam **Rupa Studio**, memungkinkan pengrajin untuk menciptakan bentuk yang sempurna dengan usaha minimal. Rilis ini juga meresmikan arsitektur 5-lapis untuk skalabilitas jangka panjang dan mengintegrasikan standar global i18n & A11y.

Mekanisme menggambar yang terinspirasi dari alat desain profesional, kini hadir di ekosistem seni piksel.

### 1. Geometric Snapping
- **Hold-to-Snap**: Tahan pointer selama 600ms di akhir tarikan untuk mengubah coretan kasar menjadi garis lurus, busur, atau lingkaran sempurna secara otomatis.
- **Perfect Circles**: Menggunakan *Midpoint Circle Algorithm* untuk menjamin simetri piksel 100% pada bentuk melingkar.
- **Axis Locking**: Secara otomatis mengunci garis ke sumbu Horizontal, Vertikal, atau Diagonal (45Â°) jika tarikan mendekati sudut tersebut.

### 2. High-Performance Rendering
- **Batch Stitching**: Seluruh tarikan garis sekarang diproses sebagai satu kesatuan (*Atomic Action*), mengoptimalkan reaktivitas DOM dan mempermudah sistem Undo/Redo.
- **SVG Pointer Track**: Jejak visual pointer yang sangat ringan dengan animasi *Marching Ants* untuk umpan balik real-time yang halus.
- **Bresenham Interpolation**: Menjamin tidak ada celah piksel (gapless) meskipun pointer digerakkan dengan sangat cepat.
