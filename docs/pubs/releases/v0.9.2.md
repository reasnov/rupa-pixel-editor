# Release Notes: v0.9.2 - The Palette & Video Precision Update

## 1. Executive Summary

**v0.9.2** focuses on expanding the creative ecosystem and perfecting technical artifacts. This release introduces a comprehensive **Palette Management** system and solves long-standing duration issues in exported video files via advanced binary header patching.

---

## 2. Key Features & Enhancements

### 2.1 Flavor Library (Palette Presets)

Manage your color ingredients with a professional-grade library system.

- **Persistent Presets:** Save custom color palettes that persist across browser sessions and projects.
- **Universal Interchange:** Import and Export recipes using the industry-standard **.gpl (GIMP Palette)** format and raw HEX lists.
- **Embedded Swatches:** Custom palettes are now automatically embedded within your `.rupa` project files for seamless sharing.
- **Quick Workflow:** Added "Add to Palette (+)" in the Color Inspector and a right-click "Spill Flavor" menu for rapid palette curation.

### 2.2 Chronos Phase 3: WebM Time Injection

Exported videos now meet professional playback standards.

- **Header Patching:** Implemented a minimalist **EBML Surgeon** (`EbmlLogic`) that injects accurate duration metadata into WebM files.
- **Perfect Seekability:** Exported videos are now fully seekable and display the correct time duration in all major media players (VLC, Premiere, QuickTime).
- **Resource Optimized:** Achieved without any external heavy dependencies, ensuring high performance on low-end hardware.

### 2.3 UI & Stability

- **Refined Sidebars:** Unified the UI patterns for Frames and Layers for a more predictable experience.
- **Timeline Scrolling:** Robust horizontal and vertical scrolling with sticky headers.
- **Bug Fixes:** Resolved initialization errors and reactivity loops in the Export and Timeline modules.

---

## 3. Blueprint & Documentation

- Synchronized technical specifications with the new Palette Interchange protocols.
- Updated documentation to reflect the retirement of the legacy Gradient tool in favor of the integrated Dither Brush.
